# Changelog

## 0.1.1 - 2026-07-18

- Adds the required action-list category icons for Stream Deck 7.5 installs.
- Adds regression coverage for category metadata and icon files.

## 0.1.0 - 2026-07-17

- First public alpha.
- Treats Mini, Neo, Plus, Classic, and XL key grids as one virtual canvas.
- Includes signal, clock, spectrum, and alignment scenes.
- Reuses unchanged tiles and supports renderer-provided dirty rectangles.
- Ships a local preview generator, tests, and a distributable plugin package.
